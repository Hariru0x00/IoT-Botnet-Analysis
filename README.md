# Simple Malware Lab

A simple personal repository for practicing analysis of **Mirai** and **Gafgyt** — two common IoT botnet families running on Linux.

## Purpose

For exclusive use in:

- Static analysis of Mirai and Gafgyt ELF binaries
- Observing dynamic behavior on isolated Linux systems

Dedicated solely to personal cybersecurity research.

## Serious Warning

- This repository may contain references or guidance related to **actual Mirai / Gafgyt samples**.
- **Absolutely do not run any binaries on a host machine or any non-isolated environment**.
- Use only within a **100% isolated Linux virtual machine** (no bridge mode, no shared folders, no port forwarding to the Internet).
