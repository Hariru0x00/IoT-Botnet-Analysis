# 4. src/ - Analysis Tool Source Code

**Purpose:** Contains custom Python scripts written to automate the analysis process.

## Files

| File | Purpose |
| :--- | :------ |
| `static_analysis.py` | Static analysis: YARA scanning, hash calculation, ELF/PE header extraction, string searching. |
| `dynamic_analysis.py` | Dynamic analysis support: API call monitoring, file activity logging, network traffic capture. |
| `__init__.py` | Marks the directory as a Python package. |
